# odin-recipes

## Description
This project is part of the Foundations Course at [The Odin Project](https://www.theodinproject.com/). The goal is to build a basic recipe website using HTML.

## Skills Demonstrated
By completing this project, I have demonstrated the following skills:
* **Git & GitHub:** Initializing repositories, branching, and using the staging/commit/push workflow.
* **HTML Basics:** Structuring a webpage with headings, paragraphs, lists, and images.
* **Linking:** Implementing both internal links (between pages) and external links.
* **File Management:** Organizing a project directory with sub folders for recipes and assets.
